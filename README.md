---
| 操作场景            | 执行命令                       | 作用说明                                       |
| :---------------: | :----------------------------: | :------------------------------------------: |
| 查看当前文件改动        | `git status`               | 检查新增 / 修改 / 删除的笔记文件                        |
| 全部文件加入暂存（含新建文件） | `git add .`                | 一次性提交所有文件夹：Algorithm、images、Projects、md 笔记 |
| 只添加单个文件         | `git add Algorithm/xxx.md` | 仅上传指定某一篇笔记                                 |
| 只添加整个文件夹        | `git add images/`          | 批量上传图片文件夹                                  |
| 生成本地提交记录        | `git commit -m "Updates"` | 引号内自定义更新备注                                 |
| 拉取远程仓库最新内容      | `git pull origin main`     | 多设备使用时先拉取，防止推送报错                           |
| 上传所有记录到 GitHub  | `git push`                 | 推送本地所有提交到远程仓库                              |
| 查看简短提交历史        | `git log --oneline`        | 快速查看过往上传记录                                 |
